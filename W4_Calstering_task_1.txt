Задание по кластеризации данных электронной коммерции
Цель
Провести кластерный анализ клиентов интернет-магазина на основе их покупательского поведения, используя датасет с Kaggle.
Датасет
Используйте датасет "E-Commerce Data" с Kaggle: https://www.kaggle.com/datasets/carrie1/ecommerce-data
Этапы выполнения задания
1. Загрузка и первичный анализ данных
· Загрузите датасет, используя библиотеку pandas.
· Проведите первичный анализ данных (размер датасета, типы данных, наличие пропусков).
· Выведите базовую статистику по числовым колонкам.
2. Предобработка данных
· Обработайте пропущенные значения (если есть).
· Удалите дубликаты (если есть).
3. Создание признаков
· Создайте новые признаки на основе существующих данных: 
o Общая сумма покупки (Quantity * UnitPrice)
o Количество уникальных товаров, купленных клиентом
o Средняя стоимость покупки
o Частота покупок (количество дней между первой и последней покупкой)
4. Нормализация данных
· Примените масштабирование (например, StandardScaler или MinMaxScaler) к числовым признакам.
5. Обработка категориальных данных
· Преобразуйте категориальные переменные (например, 'Country') в числовой формат, используя one-hot encoding или label encoding.
6. Выбор признаков для кластеризации
· Выберите наиболее релевантные признаки для кластеризации (например, частота покупок, средняя стоимость покупки, общая сумма покупок).
7. Определение оптимального числа кластеров
· Используйте метод локтя (elbow method) или силуэтный анализ для определения оптимального числа кластеров.
8. Проведение кластеризации
· Примените алгоритм K-means для кластеризации клиентов.
· Визуализируйте результаты кластеризации (например, с помощью PCA для снижения размерности до 2D или 3D).
9. Анализ результатов
· Охарактеризуйте полученные кластеры.
· Определите типичные характеристики клиентов в каждом кластере.
10. Подготовка отчета
· Подготовьте краткий отчет с описанием процесса кластеризации, полученных результатов и выводов.
11. Загрузка на GitHub
· Создайте репозиторий на GitHub.
· Загрузите все файлы проекта, включая код, отчет и визуализации.
Дополнительные задания (по желанию)
· Попробуйте другие алгоритмы кластеризации (например, DBSCAN или иерархическую кластеризацию) и сравните результаты.
· Проведите сегментацию товаров на основе их характеристик и популярности.
Критерии оценки
· Качество предобработки данных
· Обоснованность выбора признаков для кластеризации
· Корректность применения алгоритма кластеризации
· Глубина анализа полученных результатов
· Качество визуализации и отчета
· Чистота и читаемость кода
